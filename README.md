# PayLink
> Send money with just a link. Permanent on-chain receipts on Monad.

## Team
- P1: Smart Contract
- P2: Frontend Core
- P3: UX & Components

## Setup
1. Read Guide.md before writing any code
2. P1: cd contracts && forge install && forge test
3. P2: cd frontend && npm install
4. P2: Copy .env.local.example to .env.local and fill values
5. P1: forge script script/Deploy.s.sol --rpc-url monad_testnet --broadcast
